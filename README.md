# PersonalExpenseTracker

A Personal Expense Tracker application allows users to record, view, update, and delete expenses, categorize them, and set budget limits, making it ideal for demonstrating CRUD operations.

This involves managing entities like expenses, categories and budgets.

1. Expense : Users can add, view, edit, and delete expense records, including details like amount, date, and category.
2. Category : Each expense is categorized (e.g., Food, Rent, Utilities), allowing for organized financial records and easy filtering.
3. Budget : Users can set budget limits for specific categories, helping them manage spending effectively.

To build a Personal Expense Tracker that helps users monitor their finances by providing a structured system for recording income and expenses. It should allow users to
1. Create: Add new expenses, categories, and budget limits.
2. Read: View all expenses, budgets, and categories in a user-friendly interface with filtering options.
3. Update: Edit existing records, such as updating an expense amount or renaming a category.
4. Delete: Remove unnecessary or incorrect entries, helping maintain clean and accurate records.
